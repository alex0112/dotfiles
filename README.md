# Yet Another Dotfiles Repo

- Editor: I've been using [Emacs](https://www.gnu.org/software/emacs/) since I was 13.
  - Sometimes I even use it well.
  - I also mess around with [Neovim](https://neovim.io/), [Helix](https://helix-editor.com/), and [Kakoune](https://kakoune.org/)
  - And when shelling into random servers, `vim`
- Terminal: [Kitty](https://sw.kovidgoyal.net/kitty/)
  - Previously iTerm (on MacOS)
- OS/Distro/Kernel: Linux
  - Usually [Debian](https://www.debian.org/) or [Ubuntu](https://ubuntu.com/desktop)
  - FreeBSD once upon a time
  - MacOS for a while (usually because of a job)
- Shell:
  - I stick to bash these days, though I'm interested in [oils](https://oils.pub/), [elvish](https://elv.sh/), and [nushell](https://elv.sh/)
- Theme/Colors: Modified version of [Sea Shells](https://github.com/mbadolato/iterm2-color-schemes#sea-shells)
- Fonts:
  - Code related:
    - [Departure Mono](https://departuremono.com/) (I enjoy a retro, pixelated aesthetic in my terminal)
    - [0xProto](https://github.com/0xType/0xProto) (For when I want something more normal looking)
    - [FiraCode](https://github.com/tonsky/FiraCode) (Also good. However, I usually don't prefer the use of custom ligatures)
    - [Other interesting pixel fonts](https://unsung.aresluna.org/a-few-interesting-modern-pixel-fonts/)

## Other tools I find make my life and work more pleasant:
- [bat](https://github.com/sharkdp/bat) (`cat` alternative)
- [lsd](https://github.com/lsd-rs/lsd) (`ls` alternative)
- [starship](https://starship.rs/) (shell wrapper, fancy prompt)
- [fzf](https://github.com/junegunn/fzf) (search text in a fuzzy way. I **love** this one)
- [ripgrep](https://github.com/burntsushi/ripgrep) (`grep` alternative)
- [fd](https://github.com/sharkdp/fd) (`find` alternative)
- [btop](https://github.com/aristocratos/btop) (`top` alternative)
- [just](https://github.com/casey/just) (`make` alternative)
- [PaperWM](https://github.com/paperwm/paperwm) (makes GNOME shell behave like a stack-based tiling WM)
